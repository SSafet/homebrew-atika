# Atika Homebrew tap

Homebrew formulas for Atika tools.

## Install

```sh
brew install SSafet/atika/atika-bridge-desktop
```

## Pair

```sh
atika-bridge-desktop enroll --gateway https://atika.ai
```

Or visit https://atika.ai/devices to mint a pairing code.

## Formulas

- `atika-bridge-desktop` — local daemon that connects this machine to Atika so cloud agents can read/write approved project folders and run scripts.
